# Scrapp

[![Circle CI](https://circleci.com/gh/jwworth/scrapp.svg?style=svg)](https://circleci.com/gh/jwworth/scrapp)

This is an app I built to track Scrabble games.

### Setup

```
$ git clone https://github.com/jwworth/scrapp.git
$ cd scrapp
$ bundle
$ rake db:setup
$ rails s
```
